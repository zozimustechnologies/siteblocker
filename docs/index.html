Get started in 30 seconds

Install the extension

Get it from the Edge Add-ons store or load it unpacked from edge://extensions.

Open the side panel

Click the Site Blocker icon in your toolbar. The blocker opens in the browser sidebar.

Add sites to block

Type a domain (e.g., reddit.com, x.com) and click + Add. It’s blocked instantly.

Stay productive

Use toggles to pause/resume blocking. Enable the math challenge to keep kids safe. Focus on what matters.
